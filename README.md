E-Commerce Inventory Manager

Project Overview
The E-Commerce Inventory Manager is a simple command-line application built with Python to demonstrate the use of Object-Oriented Programming (OOP), functions, loops, and CRUD (Create, Read, Update, Delete) operations.
This project allows users to manage products in an inventory by adding, viewing, updating, and deleting items. Each product is assigned a unique auto-generated ID, making inventory management easier and more organized.

Features

* Add new inventory items
* View all available items
* Update existing item details
* Delete items from the inventory
* Automatically generate unique product IDs
* Menu-driven interface using a while loop
* Input validation for user interaction

Technologies Used

* Python 3
* Object-Oriented Programming (Classes and Objects)
* Functions
* Loops
* Conditional Statements
* Lists
* Dictionaries

How to Run the Project

1. Clone the repository:

git clone https://github.com/Alasko19/ecommerce-inventory-manager.git

2. Navigate to the project folder:

cd e-commerce manager

3. Run the application:

python main.py
